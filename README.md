# 데이터 품질 분석 도구

데이터 품질 분석 도구는 데이터의 정확성, 완전성, 일관성, 신뢰성, 적시성, 유효성 등을 평가하여 데이터 품질을 측정, 개선 및 관리하는 포괄적인 솔루션입니다. 이 도구는 데이터 프로파일링, 비즈니스 룰 관리, 데이터 품질 측정 및 분석, 오류 데이터 개선 활동 지원의 4가지 주요 기능을 제공합니다.

## 개요

이 웹사이트는 데이터 품질 분석의 중요성과 핵심 기능을 설명하며, 다음과 같은 주요 기능을 다룹니다:

- **데이터 프로파일링**: 데이터의 현재 상태, 구조, 내용에 대한 통계적 요약 정보를 제공
- **비즈니스 룰 관리**: 데이터 점검 규칙을 정의하고 오류 데이터를 추출
- **데이터 품질 측정 및 분석**: 품질 지표(DQI)를 기반으로 품질 현황을 분석
- **오류 데이터 개선 활동 지원**: 오류 데이터의 원인 추적 및 개선 활동을 지원

## 기능 설명

### 1. 데이터 프로파일링
- **목표**: 데이터에 대한 깊은 이해 확보 및 품질 문제 사전 식별
- **주요 기능**: 
  - 컬럼 프로파일링
  - 관계 프로파일링
  - 규칙 프로파일링
- **효과**: 잠재적 오류 사전 발견, 숨겨진 규칙 발견, 의사 결정 신뢰도 향상

### 2. 비즈니스 룰 관리
- **목표**: 데이터가 조직 정책 및 규정을 준수하도록 보장
- **주요 기능**:
  - 룰 정의 및 관리
  - 룰 적용 및 실행
  - 모니터링 및 보고
- **효과**: 비즈니스 가치 보호, 규제 준수 자동화, 현업-IT 간 소통 개선

### 3. 데이터 품질 측정 및 분석
- **목표**: 데이터 품질 수준 객관적 평가 및 개선 효과 정량화
- **주요 기능**:
  - 품질 지표 정의 및 설정
  - 품질 측정 및 분석
  - 보고서 및 대시보드
- **효과**: 객관적 품질 관리, 리스크 예방, 지속적인 개선

### 4. 오류 데이터 개선 활동 지원
- **목표**: 오류 재발 방지 및 지속적인 품질 향상
- **주요 기능**:
  - 오류 분석 및 원인 식별
  - 개선 계획 및 실행 관리
  - 효과 측정 및 피드백
- **효과**: 재발 방지, 체계적 품질 관리, 비용 절감

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 열거나 로컬 서버를 시작합니다:
   ```bash
   python -m http.server 8000
   ```
2. 각 기능별 페이지를 탐색하며 데이터 품질 분석 도구에 대한 정보를 확인합니다
